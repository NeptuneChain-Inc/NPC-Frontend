NeptuneChainAPI Documentation

Welcome to the NeptuneChainAPI documentation. This page provides information on server routes and functionalities.

Server Modules

NeptuneChainAPI utilizes various server modules including Express, dotenv for environment configuration, and additional API integrations such as database, Blockchain, Moralis, Livepeer, maps, and Stripe.

Status Codes

Status Code Description
200 Response was successful
400 Client Error: Bad Request
401 Client Error: Unauthorized
404 Client Error: Request Not Found
500 Server Error: Indicating that the request was accepted, but that an error on the server prevented the fulfillment of the request.

API Routes

Database Routes

Get User

POST /db/user/get/user

Purpose: Retrieves user information by user ID.

Parameters:

{ "uid": "string" }

Returns:

{ "user": "UserObject" }

Get Username

POST /db/user/get/username

Purpose: Retrieves a user's username.

Parameters:

{ "username": "string" }

Returns:

{ "username": "string" }

Get User Dashboard

POST /db/user/get/dashboard

Purpose: Retrieves dashboard data for a user.

Parameters:

{ "uid": "string" }

Returns:

{ "dashboard": "DashboardObject" }

Get User Videos

POST /db/user/get/media/videos

Purpose: Retrieves videos associated with a user.

Parameters:

{ "uid": "string" }

Returns:

{ "videos": "VideoObject[]" }

Get User Streams

POST /db/user/get/media/streams

Purpose: Retrieves live streams associated with a user.

Parameters:

{ "uid": "string" }

Returns:

{ "streams": "StreamObject[]" }

Set User

POST /db/user/set/user

Purpose: Create or update user data.

Parameters:

{ "userdata": "UserObject" }

Returns:

{ "result": "ConfirmationObject" }

Get Video

POST /db/media/get/video

Purpose: Retrieves video information by playback ID.

Parameters:

{ "playbackId": "string" }

Returns:

{ "video": "VideoObject" }

Get Stream

POST /db/media/get/stream

Purpose: Retrieves stream information by playback ID.

Parameters:

{ "playbackId": "string" }

Returns:

{ "stream": "StreamObject" }

Set Video

POST /db/media/set/video

Purpose: Stores video information.

Parameters:

{ "videoAsset": "VideoAssetObject", "creatorUID": "string" }

Returns:

{ "result": "ActionResult" }

Set Stream

POST /db/media/set/stream

Purpose: Stores stream information.

Parameters:

{ "streamData": "StreamDataObject", "creatorUID": "string" }

Returns:

{ "result": "ActionResult" }

Ethereum Routes

Get Signer

POST /ethereum/get/signer

Purpose: Retrieves the current Ethereum signer object.

No parameters required.

Returns:

{ "signer": "SignerObject" }

Livepeer Routes

Get Client

POST /livepeer/get/client

Purpose: Fetches Livepeer client configuration.

No parameters required.

Returns:

{ "livepeerClient": "ClientObject" }

Get Viewership

POST /livepeer/get/viewership

Purpose: Retrieves viewership statistics for a given playback ID.

Parameters:

{ "playbackId": "string" }

Returns:

{ "viewership": "ViewershipObject" }

Get Asset Metrics

POST /livepeer/get/asset_metrics

Purpose: Retrieves metrics for a given asset ID.

Parameters:

{ "assetId": "string" }

Returns:

{ "asset_metrics": "MetricsObject" }

Maps Routes

POST /maps/get/api

Purpose: Retrieves Maps API configuration.

No parameters required.

Returns:

{ "api": "APIObject" }

Moralis Routes

Get Wallet NFTs

POST /moralis/get/wallet_nfts

Purpose: Fetches NFTs from a specified wallet address.

Parameters:

{ "address": "string" }

Returns:

{ "wallet_nfts": "NFTObject[]" }

Get NFT Metadata

POST /moralis/get/nft_metadata

Purpose: Retrieves metadata for an NFT by address and token ID.

Parameters:

{ "address": "string", "tokenId": "string" }

Returns:

{ "nft_metadata": "NFTMetadataObject" }

Stripe Routes

Create Payment Intent

POST /stripe/create/payment_intent

Purpose: Creates a Stripe payment intent for handling payments.

Parameters:

{ "amount": "number", "currency": "string" }

Returns:

{ "payment_intent": "PaymentIntentObject" }

Stripe Config

POST /stripe/config

Purpose: Provides the Stripe publishable key for front-end use.

Parameters: None

Returns:

{ "publishableKey": "string" }

POST /stripe/get/price

Parameters:
