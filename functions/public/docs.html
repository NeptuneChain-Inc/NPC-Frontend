NeptuneChainAPI Documentation

Welcome to the NeptuneChainAPI documentation. This page provides information on server routes and functionalities.

Server Modules

NeptuneChainAPI utilizes various server modules including Express, dotenv for environment configuration, and additional API integrations such as database, Ethereum, Moralis, Livepeer, maps, and Stripe.

Status Codes

Status Code Description
500 Indicating that the request was accepted, but that an error on the server prevented the fulfillment of the request.
